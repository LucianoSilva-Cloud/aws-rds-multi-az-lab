# AWS RDS Multi-AZ Lab

Repositório do laboratório de implementação de banco de dados relacional gerenciado na AWS.

## 🚀 O que foi feito
* Configuração de **Security Groups** e **Subnet Groups** para isolamento de rede.
* Provisionamento de instância **Amazon RDS (MySQL)** com alta disponibilidade (**Multi-AZ**).
* Integração de uma aplicação web em **EC2** com o endpoint do banco de dados.
* Testes de persistência de dados (CRUD) em ambiente escalável.

## 🛠️ Tecnologias
* **AWS RDS** (MySQL)
* **AWS VPC** (Networking & Security)
* **AWS EC2** (Application Host)
